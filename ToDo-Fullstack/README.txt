**Client**
- Client is in event-registration-ui folder
- run 'npm install' and 'ng serve' to start client at localhost:4200

**Server**
- Server is in event-registration folder
- open /routes/Auth.js on line 12, const uri=<YOUR-CONNECTION-STRING>, set your mongodb connection string (local or MongoDB Atlas)
- repeat the same for /routes/todo.js.
- install dependencies by 'npm install'
- run by 'npm start'
